Селектори за класом (Class Selectors)

Селектори за класом дозволяють вибирати елементи з певним атрибутом class. Це дає змогу застосовувати стилі до конкретних груп елементів незалежно від їхнього типу.

  1. Ключові особливості

    • Гнучкість: Один клас можна застосувати до кількох елементів на сторінці.
    • Повторне використання: Клас можна призначати різним елементам для однотипного стилю.
    • Поєднання класів: Елемент може мати кілька класів одночасно, розділених пробілами.
    • Назва класу повинна бути унікальною і зрозумілою, щоб уникнути плутанини.
    • Не починайте назву класу з числа (наприклад, .123abc — недопустимо).
    • Не використовуйте ідентичні назви класів і тегів для уникнення конфліктів.

  2. Синтаксис

    • Клас завжди позначається крапкою . перед назвою.
    • Елементи в HTML можуть мати кілька класів, розділених пробілами.
                        .comment {
                        display: flex;
                        flex-direction: column;
                        padding: 20px;
                        border: 1px solid var(--dark-color);
                        border-radius: 15px;
                        color: #333931;
                        position: relative;
                      
                        .comment_author {
                          margin-top: 15px;
                        }
                       
                        .comment_pos {
                          margin-top: 10px;
                          color: var(--active);
                        }
                      
                        .quite_img {
                          width: 60px;
                          position: absolute;
                          top: -25px;
                          left: -25px;
                        }
                      }
