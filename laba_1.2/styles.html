1. Вбудований стиль (inline styles)

    CSS застосовується безпосередньо до елемента через атрибут style.

  2. Вбудовані стилі (internal styles)

    CSS пишеться у секції <style> у самому HTML - документі.

  3. Зовнішній файл стилів (external styles)

    CSS зберігається в окремому файлі, а HTML - документ підключає його за допомогою тегу <link>.
