Селектори за тегом (Type Selectors)

Селектор за тегом вибирає всі елементи певного типу (наприклад, всі <p>, <h1>, <div>) на сторінці. Це найпростіший і базовий вид селекторів у CSS.

  1. Ключові особливості

    • Простота: Селектори за тегом підходять для стилізації всіх елементів певного типу.
    • Масштабність: Усі елементи з однаковим тегом отримують задані стилі.
    • Низька специфічність: Стилі від селектора за тегом можуть бути перекриті стилями з більшою специфічністю (наприклад, класами або ідентифікаторами).
    • Якщо на сторінці багато однакових тегів, стилі за тегом впливатимуть на всі ці елементи, що може призвести до небажаних результатів

  2. Синтаксис

                        body {
                        font-family: 'Roboto', sans-serif;
                        font-size: 1rem;
                        line-height: 1.6;
                        background-color: var(--light-color);
                        color: --dark-color;
                        height: 100vh;
                    }
                    
                    h1 {
                        font-size: 40px;
                        width: 550px;
                        margin: 0 auto;
                        text-align: center;
                    }
