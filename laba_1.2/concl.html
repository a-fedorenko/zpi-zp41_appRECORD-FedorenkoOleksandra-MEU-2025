Висновки:

  1. Для чого використовуються каскадні таблиці стилів (CSS)?

    Каскадні таблиці стилів (CSS) використовуються для:

    • Стилізації: Зміна зовнішнього вигляду HTML-елементів (колір, шрифт, розташування, розміри тощо).
    • Відокремлення структури і дизайну: HTML відповідає за структуру контенту, а CSS — за його стиль.
    • Повторного використання стилів: Один файл CSS можна підключати до багатьох сторінок.
    • Адаптації дизайну: Створення адаптивних (responsive) веб-сторінок для різних пристроїв.

  2. Які методи визначення стилів у HTML-документі ви знаєте?

    • Вбудовані стилі (inline styles): Визначаються в атрибуті style безпосередньо в HTML-елементі.
    • Внутрішні стилі (internal styles): Пишуться в секції <style< у <head< HTML-документа.
    • Зовнішні стилі (external styles): Зберігаються в окремому файлі .css і підключаються через тег <link<.
    • CSS через фреймворки: Наприклад, використання бібліотек стилів як Bootstrap.

  3. У чому різниця між принципами каскадування і спадкування?

    • Каскадування: Це механізм вибору, які стилі будуть застосовані, якщо на один елемент діють кілька стилів одночасно. Враховується специфічність, важливість і порядок визначення стилів.
    • Спадкування: Деякі властивості (наприклад, шрифт, колір) автоматично успадковуються дочірніми елементами від батьків.

  4. Як в HTML-документі оформлюються селектори тегу і клас?

    • Селектор тегу: Вибирає всі елементи з певним тегом. 
                                  h1 {
                                color: blue;
                            }
    • Селектор класу: Вибирає елементи з атрибутом class. 
                                  .highlight {
                                color: green;
                            }

  5. Для чого використовується ідентифікатор?

    • Унікальної ідентифікації елемента на сторінці.
    • Прив'язки стилів через селектор #id: 
                                  #header {
                                text-align: center;
                                color: blue;
                            }
    • Посилання в межах сторінки через атрибут href
    • Роботи з JavaScript: 
                                  document.getElementById('header').style.color = 'red';
