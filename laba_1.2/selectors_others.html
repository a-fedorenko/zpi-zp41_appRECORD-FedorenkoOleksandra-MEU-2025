Інші селектори

  1. Селектори сусідніх елементів у CSS

    Селектори сусідніх елементів використовуються для вибору елементів, які знаходяться поруч з певним елементом. Вони дозволяють стилізувати елементи, розташовані на тому ж рівні в DOM-дереві.

    • Переваги використання
      • Зручно для стилізації структурованих даних, наприклад, списків або таблиць.
      • Зменшує потребу у створенні зайвих класів чи ідентифікаторів.
      • Дозволяє стилізувати тільки потрібні елементи залежно від їхнього розташування.
    • Безпосередній сусід (Adjacent Sibling Selector) Селектор безпосереднього сусіда вибирає перший елемент, який безпосередньо слідує за зазначеним елементом. 
                                  .category {
                                img {
                                    width: 30px;
                                    margin-right: 10px;
                                }
                            
                                img + span {
                                    font-weight: 600;
                                }
                            }
    • Загальні сусіди (General Sibling Selector) Селектор загальних сусідів вибирає всі елементи, які знаходяться на одному рівні в DOM-дереві та слідують за зазначеним елементом. 
                                  div ~ .post_card {
                                margin-right: 15px;
                            }

  2. Селектори дочірніх елементів

    Селектори дочірніх елементів використовуються для вибору елементів, які є безпосередніми дочірніми елементами певного елемента. Вони дозволяють стилізувати елементи, які знаходяться на один рівень нижче в DOM-дереві.

    • Переваги використання
      • Забезпечує більш точний вибір елементів.
      • Зменшує конфлікти стилів у випадках складної структури HTML.
      • Знижує ймовірність впливу стилів на непотрібні вкладені елементи.
    • Безпосередній дочірній елемент (Child Selector). Селектор безпосереднього дочірнього елемента вибирає всі елементи, які є безпосередніми дочірніми елементами зазначеного елемента. 
                                  .parent > .child {
                                color: blue;
                                border: 1px solid green;
                                width: 20%;
                                padding: 5px;
                                margin: 2px;
                            }
    • Нащадок (Descendant Selector). Селектор нащадка вибирає всі елементи, які є нащадками зазначеного елемента, незалежно від їхнього рівня в DOM-дереві. 
                                  .ancestor .descendant {
                                color: green;
                                border: 1px solid blue;
                                width: 20%;
                                padding: 5px;
                                margin: 2px;
                            }

  3. Селектори атрибутів

    Селектори атрибутів дозволяють вибирати HTML-елементи за наявністю або значенням певного атрибута. Це корисно для більш точного налаштування стилів.

    • Переваги використання
      • Гнучкість: Можна вибирати елементи без додавання класів чи ID.
      • Точність: Можна стилізувати елементи залежно від значення їх атрибутів.
      • Динамічність: Підходить для роботи з динамічним контентом.
    • Селектор із точним значенням атрибута. Вибирає елементи, у яких атрибут має точно вказане значення. 
                                  [input="text"] {
                                background-color: lightgray;
                            }
    • Селектор із початковим значенням атрибута. Вибирає елементи, у яких значення атрибута починається із зазначеного рядка. 
                                  [href^="https"] {
                                color: green;
                            }
    • Селектор із кінцевим значенням атрибута. Вибирає елементи, у яких значення атрибута закінчується на зазначений рядок. 
                                  [src$=".jpg"] {
                                border: 2px solid red;
                            }
    • Селектор, який містить значення атрибута. Вибирає елементи, у яких значення атрибута містить зазначений рядок. 
                                  [class*="btn"] {
                                padding: 10px;
                            }
    • Селектор із наявністю певного слова. Вибирає елементи, у яких атрибут містить слово, розділене пробілом. 
                                  [class~="highlight"] {
                                background-color: yellow;
                            }
    • Селектор із наявністю певного префіксу. Вибирає елементи, у яких значення атрибута дорівнює зазначеному значенню або починається з цього значення, розділеного дефісом. 
                                  [lang|="en"] {
                                font-style: italic;
                            }
    • Приклад 
                                  .social_item {
                                [src$=".png"] {
                                    width: 40px;
                                    cursor: pointer; 
                                }
                                [src$=".png"]:hover {
                                    transform: scale(1.1);
                                }
                            }

  4. Універсальний селектор

    Універсальний селектор (*) в CSS вибирає всі елементи на сторінці, включно з вкладеними.

    • Особливості
      • Вибір усіх елементів: Включає не лише стандартні HTML-теги (<div>, <з>, <span>), а й псевдоелементи (::before, ::after).
      • Специфічність: Універсальний селектор має найнижчу специфічність, його можна легко перекрити іншими стилями.
      • Вплив на продуктивність: Використання універсального селектора для великих проєктів може впливати на продуктивність, оскільки стилі застосовуються до кожного елемента.
