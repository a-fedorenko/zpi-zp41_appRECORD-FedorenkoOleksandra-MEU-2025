ЗАВДАННЯ

У звітному HTML-документі створити html-розміту, яка складається з наступних елементів: текст, кнопка, два інпута (поля введення). Після натискання кнопки "SWAP ME" здійснюється обмін вмістом між двома інпутами. Ви можете натиснути на неї кілька разів або вручну змінити вміст інпутів.

PROGRAM

ЗАВДАННЯ 2

JS CODE

            function swapeValue() {
                preventDefault();
    
                let inputLeft = document.getElementById("left");
                let inputRight = document.getElementById("right");
                
                const leftValue = getValue(inputLeft.name);
                const rightValue = getValue(inputRight.name);
    
                inputLeft.value = rightValue;
                inputRight.value = leftValue;
            }
    
            function preventDefault() {
                document.forms["swapeAction"].addEventListener("click", function(event){
                    event.preventDefault()
                });
            }
    
            function getValue(name) {
                return document.forms["swapeAction"][name].value;
            }

HTML

            <section class="form__section">
                <h3 class="form__title">ЗАВДАННЯ 2</h3>
                <form action="swap" name="swapeAction">
                    <button onclick="swapeValue()">SWAP ME</button>
                    <input id="left" name="left" type="text">
                    <input id="right" name="right" type="text">
                </form>
            </section>
