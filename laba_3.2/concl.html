Висновки:

Об'єкт у JavaScript — це структура даних, яка дозволяє зберігати дані у вигляді пар "ключ-значення". Ключами є строки, а значеннями можуть бути будь-які типи даних, включаючи інші об'єкти, функції чи масиви.

Варіанти створення об'єкта

  1. Через літерал об'єкта: 
                        const obj = {
                        key1: "value1",
                        key2: "value2",
                    };
  2. Через new Object(): 
                        const obj = new Object();
                    obj.key1 = "value1";
                    obj.key2 = "value2";
  3. Через клас: 
                        class Person {
                        constructor(name, age) {
                            this.name = name;
                            this.age = age;
                        }
                    }
                    const person = new Person("Іван", 30);
  4. За допомогою функції-конструктора: 
                        function Person(name, age) {
                        this.name = name;
                        this.age = age;
                    }
                    const person = new Person("Іван", 30);
  5. Створення через Object.create(): 
                        const prototype = { greet() { console.log("Привіт"); } };
                    const obj = Object.create(prototype);
                    obj.name = "Іван";

Підключення JavaScript до HTML

  1. Вбудований JavaScript: використовується тег <script> для вставки коду в HTML-сторінку. 
                        <script>
                        alert('Hello, world!');
                    </script>
  2. Зовнішній JavaScript: використовується атрибут src для підключення зовнішнього файлу з кодом. 
                        <script src="script.js"></script>
  3. Використання атрибутів подій безпосередньо в тегах HTML. 
                        <button onclick="alert('Hello, world!')">Click me</button>

Масив об'єктів

Масив об'єктів — це колекція кількох об'єктів, організованих у вигляді масиву.

            const users = [
                { id: 1, name: "Іван", age: 25 },
                { id: 2, name: "Марія", age: 30 },
                { id: 3, name: "Олег", age: 22 },
            ];

Деструктуризація об'єктів

Деструктуризація дозволяє розпаковувати властивості об'єкта в окремі змінні.

            const user = { name: "Іван", age: 25, email: "ivan@example.com" };
            const { name, age } = user;

Callback (Колбек)

Колбек — це функція, яка передається як аргумент іншій функції і викликається пізніше.

            function processUser(name, callback) {
                console.log(`Привіт, ${name}!`);
                callback();
            }
            function showSuccessMessage() {
                console.log("Дія успішно виконана!");
            }
            processUser("Іван", showSuccessMessage);

Стрілочні функції

  1. Не мають власного this: Вони успадковують значення this з батьківського контексту.
  2. Компактний синтаксис: Можна опустити фігурні дужки {} і return, якщо функція містить один вираз
            const add = (a, b) => a + b;
