Висновки:

JavaScript використовується для створення інтерактивних і динамічних веб-сторінок. Основні завдання JS в HTML-документі: JavaScript є основним інструментом для динамічних веб-сторінок. Масиви спрощують роботу з даними, а широкий набір методів дозволяє ефективно їх обробляти. Використання JS-сценаріїв надає змогу створювати інтерактивні програми, такі як калькулятори, списки та інші динамічні елементи.

  1. Виконання дій при подіях (клік, наведення курсора, введення тексту).
  2. Зміна вмісту сторінки без перезавантаження.
  3. Валідація форм (перевірка введених даних).
  4. Робота з кукісами (збереження даних на стороні клієнта).
  5. Взаємодія з сервером (отримання та відправка даних).

Підключення JavaScript до HTML

  1. Вбудований JavaScript: використовується тег <script> для вставки коду в HTML-сторінку. 
                        <script>
                        alert('Hello, world!');
                    </script>
  2. Зовнішній JavaScript: використовується атрибут src для підключення зовнішнього файлу з кодом. 
                        <script src="script.js"></script>
  3. Використання атрибутів подій безпосередньо в тегах HTML. 
                        <button onclick="alert('Hello, world!')">Click me</button>

Використання масивів у JS-сценаріях

Масиви в JavaScript дозволяють зберігати кілька значень в одній змінній. Це потужний інструмент для роботи з великими обсягами даних.

  1. Створення масиву:

    • За допомогою квадратних дужок: 
                                  let fruits = ['Apple', 'Banana', 'Cherry'];
    • За допомогою конструктора Array(): 
                                  let fruits = new Array('Apple', 'Banana', 'Cherry');

  2. Методи роботи з масивами

    • Додавання/видалення елементів: 
                                  const fruits = ['яблуко', 'банан'];
                            fruits.push('груша');    // Додати в кінець
                            fruits.pop();            // Видалити останній
                            fruits.shift();          // Видалити перший
                            fruits.unshift('апельсин'); // Додати на початок
    • Перебір масиву: 
                                  const numbers = [1, 2, 3, 4, 5];
                            numbers.forEach((num) => {
                                console.log(num);
                            });
    • Пошук елементів: 
                                  const doubled = numbers.map((num) => num * 2); // Множення всіх елементів на 2
                            const filtered = numbers.filter((num) => num > 3); // Фільтрація (тільки > 3)
    • Маніпуляції з даними: 
                                  const doubled = numbers.map((num) => num * 2); // Множення всіх елементів на 2
                            const filtered = numbers.filter((num) => num > 3); // Фільтрація (тільки > 3)
