Висновки:

  1. Адаптивна верстка

    Адаптивна верстка забезпечує зміну зовнішнього вигляду веб-сторінки в залежності від розміру екрану або пристрою, на якому вона відображається. Це досягається за допомогою медіа-запитів і спеціальних CSS-правил. Верстка з використанням медіа-запитів та стратегії Mobile First дозволяє створювати сучасні, зручні для різних пристроїв веб-сторінки. Використання метатега viewport і DevTools спрощує тестування та налагодження таких сторінок.

  2. Медіа-запити

    Медіа-запити дозволяють застосовувати різні стилі залежно від характеристик пристрою (ширини, висоти екрану, орієнтації, роздільної здатності тощо).

    CSS

                        /* Стилі для екранів ширше 768 пікселів */
                    @media (min-width: 768px) {
                        body {
                            background-color: lightblue;
                        }
                    }

                    /* Стилі для екранів менше 768 пікселів */
                    @media (max-width: 768px) {
                        body {
                            background-color: lightgreen;
                        }
                    }

  3. Медіа-типи

    Медіа-типи визначають категорії пристроїв, для яких застосовуються стилі. Основні медіа-типи:

    • all — для всіх пристроїв (за замовчуванням).
    • screen — для екранів (комп’ютерів, смартфонів, планшетів).
    • print — для друкованих пристроїв.
    • speech — для пристроїв з синтезом мови.

    CSS

                        @media screen and (min-width: 1024px) {
                        body {
                            font-size: 18px;
                        }
                    }

  4. Медіа-функції

    Медіа-функції описують характеристики пристроїв, які можна використовувати в умовах. Найчастіше використовувані медіа-функції:

    • min-width / max-width — мінімальна або максимальна ширина області перегляд.
    • min-height / max-heigh — мінімальна або максимальна висота області перегляду.
    • orientation — орієнтація екрану (portrait/landscape).
    • resolution — роздільна здатність екрана.
    • aspect-ratio — співвідношення ширини до висоти.

    CSS

                        @media (orientation: landscape) {
                        body {
                            background-color: yellow;
                        }
                    }

  5. Метатег viewport

    Метатег viewport використовується для налаштування області перегляду на мобільних пристроях. Він дозволяє масштабувати сторінку так, щоб вона виглядала коректно на різних екранах. 

                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • width=device-width: встановлює ширину області перегляду рівною ширині пристрою.
    • initial-scale=1.0: встановлює початковий рівень масштабування.

  6. Інструменти розробника

    Інструменти розробника браузера (DevTools) дозволяють:

    • Тестувати адаптивну верстку.
    • Змінювати ширину та висоту області перегляду (режим "Перегляд для пристроїв").
    • Відстежувати медіа-запити, які спрацьовують для конкретного екрана.
    • Перевіряти елементи DOM та CSS-стилі.

  7. Стратегія Mobile First

    Mobile First — підхід до створення адаптивної верстки, де спочатку проектуються стилі для мобільних пристроїв, а потім додаються стилі для ширших екранів.

      Особливості:

    • Початкова розробка для невеликих екранів.
    • Використання медіа-запитів для додавання стилів при збільшенні ширини екрану.

    CSS

                        /* Базові стилі для мобільних пристроїв */
                    body {
                        font-size: 16px;
                        background-color: lightgrey;
                    }

                    /* Стилі для ширших екранів */
                    @media (min-width: 768px) {
                        body {
                            font-size: 18px;
                            background-color: white;
                        }
                    }
